💪 RIR Guide

RIR 0
Isolations

Compounds

🔥 Block 2 Techniques (Weeks 8–12 only)
These apply ONLY to the LAST set of the exercises that show the technique badge. Weeks 1–7 = straight sets only.
Drop Sets After failure → drop weight ~25% → go to failure again → drop 25% → failure one more time. 3 back-to-back mini-sets, no rest between drops.
Example: fail at 110 → drop to 82 → fail → drop to 60 → fail.
Lengthened Partials After full-ROM failure → keep doing partial reps using ONLY the bottom/stretched portion until you can't move at all.
Example: leg curl — after full reps fail, keep doing bottom-half reps until stuck.
Myo-Reps After failure → rest exactly 5 sec → do 2 reps → rest 5 sec → 2 reps. Repeat until you can't get 2 full reps (~2–3 cycles).
Example: fail at rep 8 → 5s → 2 → 5s → 2 → 5s → 1 = done.
Weighted Static Hold Machine Chest Press only. After failure → hold in the BOTTOM (stretched) position under full tension for 30 seconds.
